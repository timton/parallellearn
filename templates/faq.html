Frequently Asked Questions


This page will fill out over time. For now, these are the only questions I can think of:

  • How to upload projects?

    For the moment, the website allows projects as Microsoft Excel files (extensions: '.xls', '.xlsx', '.xlsm', '.xltx', '.xltm'). New projects must have 2+ versions (contributing to existing projects only requires one version), each version being a different worksheet (in col. A), and all the worksheets having the same number of lines (col. A should have the same number of rows in every worksheet). The aligning (making sure that all the lines correspond to their translation, based on the row number) is done manually by the author of the project.

  • What about Copyright?

    The website is purely educational, nobody is looking to make any money with it - any resources uploaded here are used for learning purposes. For the moment, I will pay for the web-hosting services myself. Additinally, when uploading a project, there's a special field for indicating the source of each and every version - this way we make sure to credit any work done by somebody else.
